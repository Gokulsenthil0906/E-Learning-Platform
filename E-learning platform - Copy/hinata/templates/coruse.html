Cloud Computing

This course provides an overview of the concepts and technologies of cloud computing, a paradigm that enables on-demand access to shared and scalable computing resources over the internet. Students will learn about the benefits and challenges of cloud computing, as well as the main service and deployment models, such as Infrastructure as a Service (IaaS), Platform as a Service (PaaS), Software as a Service (SaaS), and public, private, and hybrid clouds.
